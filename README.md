## 🔐 Security & Privacy

- **Zero-knowledge architecture** — we don't store access codes in plain text
- **Zero-log policy** — no IP addresses, no metadata, no activity tracking
- **No passwords** — 32-character access codes are hashed with bcrypt
- **HTTPS only** — TLS 1.2/1.3, HSTS enabled
- **Rate limiting** — brute-force protection on all endpoints
- **XSS/CSRF protection** — input sanitization, security headers
- **Spam filtering** — automatic detection and quarantine
