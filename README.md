## Security & Privacy Architecture

- **Zero-knowledge architecture** — access codes hashed with bcrypt, never stored in plain text
- **Zero-log policy** — no IP addresses, no metadata, no activity tracking of any kind
- **No passwords** — 32-character cryptographic codes eliminate phishing and password leaks
- **HTTPS only** — TLS 1.2/1.3 with HSTS preload (2-year max-age)
- **Rate limiting** — brute-force protection on all authentication endpoints
- **Security headers** — X-Content-Type-Options, X-Frame-Options, CSP, Referrer-Policy
- **Spam filtering** — automatic detection with scoring and quarantine
- **OCSP Stapling** — faster TLS handshakes
- **No cookies** — no tracking cookies, no third-party scripts
